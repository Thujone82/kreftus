# PWA Icons

This directory should contain PWA icons for the Forecast app:

- `icon-192.png` - 192x192 pixels (required)
- `icon-512.png` - 512x512 pixels (required)

These icons will be used for:
- PWA installation
- Home screen icons
- Splash screens

You can create these icons using any image editor. The icons should represent the Forecast weather app.
