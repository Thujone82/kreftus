CLI Applications

Larry v1.1

Larry is a terminal Frogger-like game written in Go with tcell. Guide a green '@' across traffic with WASD/Arrows, Space to pause, Esc to quit. Levels change color themes, add vehicles, and increase speed; safe shoulders and gaps help you through. Includes a persistent Top-10 scoreboard with name entry. Downloads and README on the project page.

GetForecast

GetForecast is a PowerShell script that retrieves and displays detailed weather information for a specified location using the National Weather Service API. It accepts US zip codes or "City, State" strings, and provides current conditions, hourly forecasts, daily forecasts, weather alerts, and more with smart color-coding and interactive display modes. Features include geocoding via OpenStreetMap Nominatim API, temperature trend detection, wind gust information, dew point calculations, and an interactive mode with keyboard controls. No API registration required.

vBTC v1.5

vBTC is a command-line application for simulating Bitcoin trading. It uses a live API to fetch real-time Bitcoin prices and allows users to manage a virtual portfolio with a starting capital of $1000. The application features real-time market data, including 24h high/low, volatility, and a 1-Hour Simple Moving Average (SMA), with a 15-minute cache to optimize API calls. It provides a full transaction ledger, configuration options, and color-coded feedback for an intuitive trading experience.

Bmon v1.5

It features multiple monitoring modes, including an interactive mode with start/pause functionality, a 5-minute quick-glance mode (-go), and a 24-hour long-duration single-line mode (-golong). Features include a visual sparkline chart showing price trends, audible price-change alerts, and on-demand currency conversions (BTC ‚Üî USD ‚Üî Satoshis). Available in both Go and PowerShell versions, providing cross-platform compatibility and native performance.

GetWeather CLI

GetWeather CLI is a command-line utility for retrieving detailed weather information for any specified location. It accepts US zip codes or "City, State" strings and uses the OpenWeatherMap API to provide current conditions, forecasts, and alerts directly in your terminal. It is available as both a PowerShell script and a native Go application.

UPDate Tool

UPDate v1.1 is an interactive PowerShell command-line tool for managing and executing file update jobs. It allows you to configure update sources (local paths or URLs) and destination directories, then execute updates with a simple interactive interface featuring color-coded output and multiple job selection capabilities. Features include Esc key navigation, job removal functionality, enhanced auto mode, and improved output display with Source/Destination information. rc (Run Continuously)

rc v1 is a command-line utility that executes a given command string in a loop. After each execution, it waits for a specified number of minutes before running the command again. It features an optional precision mode to account for the command's execution time, preventing timing drift and ensuring runs start at a consistent interval.

Available in a native Go version for cross-platform performance and a PowerShell version for Windows environments.
