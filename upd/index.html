UPDate Tool

UPDate is an interactive PowerShell command-line tool for managing and executing file update jobs. It allows you to configure update sources (local paths or URLs) and destination directories, then execute updates with a simple interactive interface featuring color-coded output and multiple job selection capabilities.

Features include interactive CLI with color-coded output, multiple job selection and execution, support for local file copying and URL downloads, JSON-based configuration storage, command-line arguments for automation, and verbose mode for debugging. The tool supports both interactive and automated execution modes with enhanced navigation using Esc key functionality.
