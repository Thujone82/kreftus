GoLPy v1.0 — Game of Life

GoLPy v1.0

GoLPy is a pygame desktop port of the Kreft.us web Game of Life. Run classic Conway B3/S23 cellular automata with colorful cell aging, 57 built-in patterns, and full playback controls. Also available as a browser PWA at gol/.

Features:

  • Terminal UI: -tui or gol-tui.exe — setup screen then full-terminal simulation. Press C for controls help.
  • Wrapped mode: Toroidal grid sized to the window (pygame) or terminal (-tui). Resize before play to change grid shape; edges wrap like the web edition.
  • Infinite mode: Sparse unbounded grid for spaceships, guns, and large patterns.
  • 57 patterns: Gliders, oscillators, Gosper gun, methuselahs, spacefillers, and more.
  • Playback: Play/pause, single-step, reset, and adjustable speed.
  • Navigation: Pan and zoom when paused (wrapped pygame) or anytime (infinite); click cells to toggle. -tui: arrows/HJKL pan in infinite mode (paused, or while running with follow off via F).
  • Memory: M+ / MR in-memory save and restore.
  • CLI: --mode, --pattern, --speed, -tui, -debug. Use --help for key bindings (gol-tui.exe --help is TUI-only).

Run from source with python gol.py or python gol_tui.py, or use the standalone builds below.
