BTC - Bitcoin Price Tracker & Portfolio Monitor

BTC v2.2

BTC is the CLI version of BTC Track, a PowerShell script for Bitcoin price tracking and portfolio monitoring. It retrieves real-time and historical Bitcoin data from the LiveCoinWatch API, displaying comprehensive market information and portfolio metrics.

Key features include:

  • Real-time Bitcoin price with accurate 24-hour price difference calculation
  • Portfolio tracking with profit/loss calculation in USD and percentage
  • 1H SMA (Simple Moving Average) with color-coded indicators
  • 24H high/low prices with timestamps
  • Volatility percentage showing price swing over 24 hours
  • 24H volume and market capitalization
  • History sparkline visualization showing 24-hour price trend
  • CSV logging for historical data tracking
  • Interactive first-run setup and configuration management
  • Color-coded output for intuitive price movement visualization
  • Verbose mode for detailed operational messages

The script uses a configuration file (btc.ini) to store API keys, portfolio settings, and logging preferences. Command-line parameters can override configuration settings for individual runs.
