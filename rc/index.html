rc - Run Continuously

rc v1.4

rc (Run Continuously) is a command-line utility that executes a given command string in a loop. After each execution, it waits for a specified number of minutes before running the command again.

Features:

  • Precision Mode (-p): Accounts for the command's execution time, preventing timing drift and ensuring runs start at a consistent interval.
  • Period Suffixes: Support for time units: 's' (seconds), 'm' (minutes, optional), 'h' (hours). Integers without suffix default to minutes. Examples: 5, 15s, 5m, 1h.
  • Skip Mode (-skip): Skip a specified number of initial executions before starting command execution. If -skip 0 is specified, it defaults to 1.
  • Limit Mode (-limit): Limit the total number of executions to perform. Skipped executions do not count toward this limit.
  • Silent Mode (-s): Suppresses status output messages while still displaying command output and errors.
  • Clear Mode (-c): Clears the screen before executing the command in each iteration for clean output display.

Available in a native Go version for cross-platform performance and a PowerShell version for Windows environments.
