Bmon - Bitcoin Price Monitor

Bmon v1.5

Bmon is a lightweight, real-time Bitcoin price monitor that runs in the command line. It features multiple monitoring modes, including an interactive mode with start/pause functionality, a 5-minute quick-glance mode (-go), and a 24-hour long-duration single-line mode (-golong).

Key features include:

  • Real-time price updates with configurable refresh intervals
  • Visual sparkline chart showing price trends
  • Audible price-change alerts
  • On-demand currency conversions (BTC ↔ USD ↔ Satoshis)
  • Color-coded price changes for intuitive monitoring

Available in both Go and PowerShell versions, providing cross-platform compatibility and native performance.
