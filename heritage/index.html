🪴 PDX Heritage Trees

Welcome to PDX Heritage Trees

A local field guide to Portland’s registered Heritage Trees. Explore a map of every tree on the City registry, mark the ones you have visited, and take notes — all stored privately in your browser.

What this app does

  • Loads the full list of Portland’s Heritage Trees from a bundled snapshot.
  • Looks up each tree’s coordinates so it can be plotted on a map.
  • Lets you mark trees as Found, timestamp the find, and jot notes.
  • Finds the 10 closest trees to you and can launch walking directions.
  • Stays up to date when the City adds or removes trees — your notes are preserved.

You’ll need a free Google Maps API key

  1. Open the Google Cloud Console → Credentials and sign in.
  2. Create a new project (or select an existing one) and click Create Credentials → API Key.
  3. Enable these APIs on the project:
    • Maps JavaScript API – for the map and markers
    • Geocoding API – to turn tree addresses into coordinates
  4. (Recommended) Restrict the key to the two APIs above and to your site’s URL.
  5. Paste the key below. Google’s free monthly credit covers the one-time geocoding of ~400 trees and normal map use by a long way.

Your key is stored in your browser’s localStorage. It never leaves your device except in the requests it makes directly to Google.
